This is the first post on the blog. It's here to make sure everything works.

## What this blog is

A place to write things down. Nothing fancy — just markdown files that get turned into HTML.

## How it works

Posts live in the `posts/` directory as `.md` files. Run `python build.py` and the whole site gets generated into `_site/`.

> Simple is better than complex.

Here's an example of a code block:

```
python build.py
```

That's it for now.
